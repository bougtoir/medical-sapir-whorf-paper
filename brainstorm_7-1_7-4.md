# ブレインストーミング: セクション 7-1, 7-4（研究アジェンダ）

## 7-1: Before-after diagnostic pattern analysis（診断パターンの前後比較分析）

### 現状の記述
ICD-11採用医療システムの行政クレームデータを用い、移行前後の慢性疼痛診断パターンを比較する。

### 注意点
- 多くの国でICD-11への移行は未完了のため、前後比較は現時点では実施困難
- 実現可能性の高いデータソースの選定が鍵

### データソース候補

| 国 | 状況 | データベース | 備考 |
|----|------|-------------|------|
| **タイ** | 2022年に世界初でICD-11を全国採用 | 国民皆保険行政データベース | **現時点で唯一前後比較が可能な国** |
| **日本** | ICD-11移行準備中 | DPC/PDPS、NDB（レセプトデータベース） | 移行前ベースラインの取得好機 |
| **韓国** | ICD-11移行検討中 | HIRA（健康保険審査評価院）データベース | |
| **ドイツ** | ICD-10-GMからの移行過程 | Barmer等保険者データ | |

### 「診断マイグレーション」の追跡
- ICD-10でR52（疼痛、詳細不明）やG89（疼痛関連コード）に分類されていた患者が、ICD-11移行後にMG30.0〜MG30.6のどこに移動するか？
- 仮説：R52の大部分がMG30.0（慢性一次性疼痛）に「診断的昇格」する → これ自体が治療的命名の実証
- 「未分類→分類済み」への移行率が、疾患命名の治療的効果の代理指標になりうる

### アウトカムの拡張案

**一次アウトカム**（現行）
- 診断分布、特異的/非特異的診断の割合、専門医紹介パターン

**追加候補**
- オピオイド処方パターンの変化（命名により非薬物療法への移行が起きるか）
- 救急外来受診率（診断名獲得により計画的受診が増えるか）
- 診療継続率（dropout rateの変化）
- 医療費の変化（短期的には増加するが、長期的には不必要な検査の減少により削減される仮説）
- 精神科併存診断率の変化（「原因不明の痛み」→「慢性一次性疼痛」で精神科紹介が減るか）

### 研究デザインの精緻化
- **中断時系列分析（Interrupted Time Series; ITS）**: ICD-11導入時点を介入点とし、トレンドの変化を検出。最低24ヶ月の前後データが望ましい
- **コントロール条件**: 疼痛以外の疾患（例：糖尿病、高血圧）の診断パターンを同時に追跡し、ICD-11移行の一般的効果と疼痛特異的効果を分離
- **サブグループ解析**: MG30.0〜MG30.6の各カテゴリー別に分析。特にMG30.0（慢性一次性疼痛）はICD-10に対応カテゴリーが存在しないため、最もドラマチックな変化が予測される
- **感度分析**: コーディング研修の影響を統制するため、導入直後（学習期間）と安定期を分けた分析

### 日本特有の強み
- DPC/PDPSデータは病名と処方・処置が紐づいており、「命名→治療行動」の因果連鎖を直接観察できる
- NDBは全国悉皆データであり、地域差の分析も可能
- 日本のICD-11移行は段階的に進むため、自然実験としての設計に有利

## 7-4: Research funding analysis（研究資金配分分析）

### 現状の記述
ICD-11採用前後の慢性疼痛研究の出版物・資金配分の計量書誌学的分析。予測：新規ICD-11コードを得たカテゴリーが研究活動の不均衡な成長を示す。

### 「命名→研究爆発」の最も鮮明な先行事例：nociplastic pain
- 2017年のIASPによるnociplastic pain定義以前/以後のPubMed検索ヒット数を時系列で示す
- 2017年以前：「central sensitization syndrome」「functional pain」等の散逸的用語
- 2017年以後：「nociplastic」という統一用語に収束 → 研究コミュニティの形成を促進
- これをNTLモデルの「N→R」段階（命名→研究投資）の実証例として位置づけ

### データソース候補

| カテゴリー | データソース | 用途 |
|-----------|-------------|------|
| 出版物 | PubMed/MEDLINE、Web of Science、Scopus | 論文数・引用数推移 |
| MeSH terms | NLM MeSH Browser | ICD-11対応のMeSH追加の追跡 |
| 研究資金（米国） | NIH RePORTER | 慢性疼痛関連グラント推移 |
| 研究資金（日本） | KAKEN（科研費DB） | キーワード別研究費推移 |
| 研究資金（EU） | Horizon Europe | 欧州の資金動向 |
| 臨床試験 | ClinicalTrials.gov、UMIN-CTR | 登録数推移 |
| 学会活動 | IASP、日本ペインクリニック学会 | セッション・演題数推移 |
| 特許 | Google Patents、J-PlatPat | 疼痛関連特許出願の変化 |

### 分析手法の候補

1. **計量書誌学的マッピング**: VOSviewerやCitNetExplorerを用いたキーワード共起ネットワークの時系列変化。ICD-11導入前後でクラスター構造がどう変わるか
2. **引用ネットワーク分析**: Treede et al. (2015)やNicholas et al. (2019)等の分類提案論文からの引用爆発パターン
3. **Difference-in-Differences**: ICD-11で新規カテゴリーを得た疼痛領域（MG30.0: 慢性一次性疼痛）vs. 既存カテゴリーの再編にとどまった領域（MG30.5: 慢性神経障害性疼痛）の研究成長率を比較
4. **「名前なきもの」の消失パターン**: 以前は「原因不明の慢性疼痛」として扱われていた研究が「慢性一次性疼痛」に再分類される過程を追跡

### 仮説の精緻化

- **H1（量的効果）**: ICD-11で完全に新しいカテゴリーを得た領域（MG30.0, MG30.2）は、既存カテゴリーの再編にとどまった領域より、出版物数の成長率が有意に高い
- **H2（質的効果）**: 新カテゴリー導入後、当該領域のRCT（ランダム化比較試験）の割合が増加する（観察研究→介入研究への移行＝研究の成熟を示す）
- **H3（マシュー効果）**: 初期に高引用を得た分類論文の著者群が、後続の研究資金を不均衡に獲得する（科学社会学的な「名づけの特権」）
- **H4（産業波及効果）**: 新疼痛カテゴリーに対応する臨床試験の製薬企業スポンサー率が、カテゴリー確立後に上昇する

### 日本のKAKENデータの活用
- KAKEN（科研費データベース）は研究費の詳細が公開されており、キーワード検索で「慢性疼痛」「nociplastic」等の研究費推移を追跡可能
- 日本ペインクリニック学会の学術大会演題のテキストマイニングも可能
